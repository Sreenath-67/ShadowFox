1. CHECK SYSTEM IP ADDRESS
Command:
ip a

Description:
Used to find the IP address of the system (target machine).


2. BASIC PORT SCAN

Command:
nmap <target-ip>

Example:
nmap 192.168.56.101

I used:
nmap 192.168.2.128

Description:
Scans the most common 1000 ports to identify open ports.


3. SCAN ALL PORTS

Command:
nmap -p- <target-ip>

Example:
nmap -p- 192.168.56.101

I used:
nmap -p- 192.168.2.128

Description:
Scans all 65535 ports on the target system.

4. SERVICE VERSION DETECTION

Command:
nmap -sV <target-ip>

Example:
nmap -sV 192.168.56.101

I used:
nmap -sV 192.168.2.128

Description:
Detects the version of services running on open ports.

5. AGGRESSIVE SCAN

Command:
nmap -A <target-ip>

Example:
nmap -A 192.168.56.101

I used:
nmap -A 192.168.2.128

Description:
Performs advanced scan including:
- OS detection
- Service detection
- Script scanning
- Traceroute
